1. \(25+(x^2+y^2+z^2)\) is a positive function on all of \(S\), and so \(\displaystyle \int\int_{S}25+(x^2+y^2+z^2) \,dS\) is positive.

  2. \(25-(x^2+y^2+z^2)\) is exactly equal to zero on all of \(S\), and so \(\displaystyle \int\int_{S}25-(x^2+y^2+z^2) \,dS\) is zero.

  3. \(25-(2x^2+2y^2+2z^2)\) is negative on all of \(S\), and so \(\displaystyle \int\int_{S}25-(2x^2+2y^2+2z^2) \,dS\) is negative.
