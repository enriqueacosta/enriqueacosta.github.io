Solution 1.2.3.1.

Any pair of values \((x,y)\) which is a solution must satisfy both equations. In particular, the \(y\)-coordinates of any solution have to both equal \(x^2+3x-5\) and \(x+3\text{.}\) Thus, we have

\begin{equation*} x^2+3x-5=x+3, \end{equation*}

which has solutions \(x=-4\) and \(x=2\text{.}\) These are the only values of \(x\) for which \((x,y)\) can satisfy both equations in the system of equations. Setting \(x=-4\) in either of the two equations we find \(y=-1\text{,}\) so \((x,y)=(-4,-1)\) is a solution (the only one with \(x=-4\)). Setting \(x=3\) in either of the two equations we find \(y=5\text{,}\) so \((x,y)=(2,5)\) is the only other solution.

Thus, the only solutions are \((x,y)=(-4,-1)\) and \((x,y)=(2,5)\)
