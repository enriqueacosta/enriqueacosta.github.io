A critical point of a function \(g(x,y)\) is a point where \(\nabla g(x,y)=\vec{0}\) or where \(\nabla g(x,y)\) is undefined. The partial derivatives

\begin{equation*} \begin{aligned} g_x(x,y)\amp =10x+4y+4\\ g_y(x,y)\amp = 4x-4y-4, \end{aligned} \end{equation*}

show that \(\nabla g(x,y)\) is defined everywhere, and \(\nabla g(x,y)=\vec{0}\) only at the points which are solutions the system of equations

\begin{equation*} \begin{aligned} 10x+4y+4 \amp = 0\\ 4x-4y-4 \amp = 0. \end{aligned} \end{equation*}

If we add these two equations together, the \(4y\) will cancel giving \(14x=0\text{,}\) so \(x=0\text{.}\) Setting \(x=0\) in either of the two equations gives \(y=-1\text{.}\) Thus, the only solution to the system is \((x,y)=(0,-1)\text{.}\) Thus, the only critical point is \((0,-1)\text{.}\)
