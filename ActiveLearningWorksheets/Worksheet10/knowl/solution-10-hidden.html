Solving for \(y\) in the second equation gives \(y=-x^2/2\text{.}\) Any pair of values \((x,y)\) which is a solution to the system must satisfy \(y=-x^2/2\text{,}\) and so we can plug this into the first equation, giving

\begin{equation*} \begin{aligned} 2x\left(-\frac{x^2}{2} \right)+\left(-\frac{x^2}{2} \right)\amp =\amp 0 \\ -x^3-\frac{x^2}{2}\amp =\amp 0\\ -\frac{1}{2} x^2 (2 x + 1)\amp =\amp 0 \end{aligned} \end{equation*}

so \(x=0\) or \(x=-1/2\text{.}\) These are the only two possible values of \(x\) for which \((x,y)\) may be a solution of the system (for some \(y\)). Setting \(x=0\) in the second equation of the system we find \(y=0\text{,}\) so \((x,y)=(0,0)\) is a solution (the only one with \(x=0\)). Setting \(x=-1/2\) in the second equation of the system we find \(y=-1/8\text{,}\) so \((x,y)=(-1/2,-1/8)\) is the only other solution.
