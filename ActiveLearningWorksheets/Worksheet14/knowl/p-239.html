We compute \(D\)

\begin{equation*} D=\det \pmatrix{f_{xx} \amp f_{xy} \\ f_{yx} \amp f_{yy}}=\det \pmatrix{-2 \amp 0 \\ 0 \amp -8}=16, \end{equation*}

and see that \(D\gt 0\) (at any point so in particular at \((-1,1/2)\)), so we see that \((-1,1/2)\) is a local maximum or a local minimum. To figure out which one it is, we can look at, for example, the concavity in \(x\)-direction by computing \(f_{xx}(-1,1/2)\) which gives \(f_{xx}(-1,1/2)=-2\lt 0\text{.}\) This indicates that \(f\) is concave down in the \(x\)-direction at \((-1,1/2)\) and so \((-1,1/2)\) is a local maximum.
