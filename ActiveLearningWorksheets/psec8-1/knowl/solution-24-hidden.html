1. Since the grade of the ramp is \({7} \% = {7}/100 \text{,}\) this means that a \({7}\)-foot height difference occurs over a horizontal distance of 100 feet. So we have \(\tan \theta = \frac{{7}}{100}.\) Using the \(\tan^{-1}\) button on the calculator, we get

    \begin{equation*} \theta = \tan^{-1}\left(\frac{{7}}{100}\right) = 4.00417 ^\circ. \end{equation*}

  2. From the right triangle representing the ramp we see that one leg represents the height difference between the path and the deck, which is \(h={1.5}\) ft. The other leg represents the path, of which we would like to find the length \(x\text{.}\) Then \({1.5}/x = \tan 4.00417^\circ\)Solving this equation for \(x\text{,}\) we have

    \begin{equation*} x = \frac{{1.5}}{\tan 4.00417^\circ} = 21.429. \end{equation*}

    So the path has to be 21.429 feet long. (We can also use similar triangles: \({1.5}/{7} = {x}/{100}\text{.}\))

  3. The ramp is represented by the hypotenuse \(y\) of the right triangle. Using the Pythagorean theorem we have

    \begin{equation*} \begin{aligned} y \amp= \sqrt{a^2+{1.5}^2} \\ \amp= 21.481 \text{ feet}. \end{aligned} \end{equation*}

    (We can also use \(\sin4.00417^\circ= {1.5}/y\text{.}\))
