Item 6.3.b.

From the right triangle representing the ramp we see that one leg represents the height difference between the path and the deck, which is \(h={1.5}\) ft. The other leg represents the path, of which we would like to find the length \(x\text{.}\) Then \({1.5}/x = \tan 4.00417^\circ\text{.}\) Solving this equation for \(x\text{,}\) we have

\begin{equation*} x = \frac{{1.5}}{\tan 4.00417^\circ} = 21.429. \end{equation*}

So the path has to be 21.429 feet long. (We can also use similar triangles: \({1.5}/{7} = {x}/{100}\text{.}\))
