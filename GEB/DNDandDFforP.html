GEB: Figure and Ground

What is this?

This is an interactive playground for the DND (Does-Not-Divide) and DF (Divisor-Free) formal systems described in Chapter III of Douglas Hofstadter's Gödel, Escher, Bach: an Eternal Golden Braid.

These systems demonstrate how complex mathematical properties, like whether a number is prime, can emerge from simple, mechanical rules of string manipulation (adding dashes), without any built-in knowledge of arithmetic.

The DND-System

Rules:
1. Axiom: xy DND x (a number larger than x does not divide x).
2. Rule: If x DND y, then x DND xy (adding multiples of x to a number that x does not divide will not change the fact that x does not divide).
